# Magisterium — сайт для подготовки

Интерактивный сайт для самостоятельной подготовки к вступительному тестированию в магистратуру.

## Режимы

- **Экзаменационный режим** — случайная выборка заданий, итоговый результат после завершения.
- **Тренировка с ответами** — результат каждого ответа показывается сразу.

## Блоки

| Блок | Банк заданий | В одной попытке |
| --- | ---: | ---: |
| Английский язык (грамматика) | 500 | 50 |
| ТГО | 500 | 30 |
| M095: программирование и ИБ | 500 | 40 |

## Запуск

Откройте `index.html` в браузере. Сайт полностью автономный: его можно переместить на другой компьютер, загрузить в облачное хранилище или разместить на статическом хостинге. Внешние папки с учебными материалами для работы сайта не требуются.

При наличии Python можно запустить локальный сервер из папки сайта:

```powershell
python -m http.server 8000
```

После этого откройте `http://localhost:8000`.
